# Venall
Venall — это монолитное Laravel-приложение, объединяющее три взаимосвязанных pet-проекта в единую систему с общей авторизацией и дизайном.
Пока в нем только 3 пет проекта: To-Do лист с системой магазина, блог с системой лайков и комментариев и Дашборд в котором вся полезная информация, а так же присутствует авторизация и главаня страница с выбором проекта.
Чтобы запустить проект скачайте проект на устройстве должен быть PhP и Composer
Команды чтобы запустить проект:
- ```composer install``` - чтобы установить все зависимости для Laravel
  - Если выдастся ошибка связанная с ```openssl``` нужно изменить файл ```php.ini``` изменить в нем строчку с ```;extension = openssl```  на ```extension = openssl``` то есть убрать ;
- ```copy env.example .env``` - копировать файл emv.example в .env
- Так же нужно настроить БД в .env я использовал posgrae
- ```php artisan key:generate``` - сгенерировать ключ приложения
- ```php artisan migrate``` - выполнить миграции к БД
- ```php artisan serve``` - запустить локальный сервер и перейти по указаноой ссылки.
- Так же чтобы просмотреть функционал сайта нужно зарегестрироваться и после авторизироваться.
