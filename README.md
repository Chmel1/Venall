# Venall
Большой монолитный пет проект
Пока в нем только 2 пет проекта: To-Do лист с системой магазина и Обычный блог с системой лайков и комментариев а так же присутствует авторизация и главаня страница с выбором проекта.
Чтобы запустить проект скачайте проект и используйте команду в папке проекта через командную строку ```php artisan serve``` и перейдите по локальной ссылке указаной в командной строке.
